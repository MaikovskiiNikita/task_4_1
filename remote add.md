## git remote add

**git remote add *[имя_удаленного_репозитория][адрес]*** - Добавление удалённого репозитория.

Чтобы добавить удалённый репозиторий, используйте команду: 

```bash=
git remote add
```
> Указав shortname и url требуемого репозитория.
